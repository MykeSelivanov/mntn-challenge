# API tests
Api test for https://reqres.in

## Technologies
- JavaScript
- Node
- Chai 
- Chai-http 
- Mocha 

## Usage
To run on local ->
open directory in terminal
- before running, install dependencies
```bash
npm i
```
- to run (this will run mocha)
```bash
npm test
```

## Test results
Mocha will output test results in CLI
